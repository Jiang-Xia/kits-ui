# Waterfall API

## Waterfall 属性

| 属性名     | 说明         | 类型      | 可选值       | 默认值 |
| ---------- | ------------ | --------- | ------------ | ------ |
| isAnimated | 是否开启动画 | `boolean` | false / true | true   |
| type       | 类型         | `string`  | text / img   | text   |
| width      | 宽度         | `string`  |              | 300px  |
| height     | 高度         | `string`  |              | 50px   |
| radius     | 圆角         | `string`  |              | 0      |
